# Generated acados Solver Code

This directory contains C code generated by the acados OCP solver framework.

## How to Generate

1. Install acados and CASAdi:
   ```bash
   pip install casadi
   # Follow https://docs.acados.org/installation/ for acados
   export ACADOS_DIR=/path/to/acados
   ```

2. Run the codegen script:
   ```bash
   cd workspace/robotics/control/mpc/tools
   python generate_solver.py
   ```

3. Rebuild the project:
   ```bash
   cmake -B build -S workspace -DCMAKE_BUILD_TYPE=Release
   cmake --build build -j$(nproc)
   ```

## Without acados

If acados is not installed, the MPC module automatically falls back to an
Eigen-based LTV-MPC solver. This is always available and requires no
additional dependencies.
