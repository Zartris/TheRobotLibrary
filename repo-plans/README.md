This folder is for the bigger picture. We are placing the larger milestones here, and the smaller ones will be placed in the respective folders. This is to keep track of the progress and to have a clear view of the roadmap.

fx. First milestone: "Build a mobile robot simulation in 2d that enables all the basic functionalities of a mobile robot, such as localization, mapping, and navigation. This will be the foundation for all the other milestones, and it will allow us to test and validate our algorithms in a controlled environment."

So in the controller plans folder we have only task related to how we expand the controller, and in the perception plans folder we have only task related to how we expand the perception. This way we can keep track of the progress in each field and we can also see how they are progressing in relation to each other. This is also enabling us to have a clear view of the roadmap and to see how we are progressing towards our goals and enables parallel development in different fields.
